type contents = [ `Case of
           ((t * string) list * contents) list
       | `String of string
       | `Strings of string list ]
Contents can be either a string or case conditions. The handler case (v1,c1) ... (vn,cn) * action is the case enabled cases where all variables vi are equal to ci. In that case the action action is performed.
